<h1>E-COMMERCE PRODUCT RECOMMENDATION SYSTEM</h1>

A  product recommendation system built using Python, Pandas, Scikit-Learn, and machine learning. It recommends similar items by analyzing product features and similarity scores.

<h2>FEATURES</h2>

Recommends similar products based on feature similarity

Uses clustering + cosine similarity

Fast, simple, and easy to extend

Works fully on preprocessed product data

<h2>TECHNOLOGIES USED</h2>

<b>Python</b>

<b>Pandas</b>

<b>Scikit-Learn</b>

<b>Machine Learning</b>
